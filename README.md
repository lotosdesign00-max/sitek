# Rival Design — Premium Multi-Page Website

Профессиональный многостраничный сайт для дизайн-бренда Rival Design.

## 🎨 Концепция

**"Premium Cosmic Design Command Center"**
- Темная космическая эстетика с красными акцентами
- Футуристичный, строгий, технологичный дизайн
- Премиум-позиционирование
- Специализация на аватарах

## 📁 Структура сайта

```
├── home.html              # Главная страница
├── pages/
│   ├── services.html      # Услуги
│   ├── portfolio.html     # Портфолио
│   ├── about.html         # О нас
│   ├── process.html       # Процесс работы
│   ├── testimonials.html  # Отзывы
│   ├── contact.html       # Контакты (конверсия)
│   └── faq.html          # FAQ
├── css/
│   └── main.css          # Главные стили
└── js/
    └── main.js           # Интерактивность
```

## 🚀 Страницы

### 1. Home (home.html)
- Hero с космическим фоном и частицами
- Позиционирование и ценности
- Флагманские услуги
- Избранные кейсы
- Доверие и отзывы
- CTA секции

### 2. Services (pages/services.html)
- Полный список услуг
- Категории: Флагманские, Статика, Анимация, Пакеты
- Детальное описание каждой услуги

### 3. Portfolio (pages/portfolio.html)
- Кураторская подборка проектов
- Кейсы с результатами
- Категории проектов

### 4. About (pages/about.html)
- История бренда
- Философия дизайна
- Подход к работе
- Специализация на аватарах

### 5. Process (pages/process.html)
- 5-шаговый процесс работы
- Прозрачность и профессионализм
- Снижение барьеров

### 6. Testimonials (pages/testimonials.html)
- Отзывы клиентов
- Социальное доказательство
- Доверие

### 7. Contact (pages/contact.html)
- **Главная конверсионная страница**
- Telegram как основной канал
- Контактная форма
- Инструкции для первого сообщения

### 8. FAQ (pages/faq.html)
- Частые вопросы
- Снятие возражений
- Прозрачность процесса

## 🎯 Конверсионная стратегия

### Путь пользователя:
1. **Home** → Впечатление + интерес
2. **Services** → Понимание возможностей
3. **Portfolio** → Доказательство уровня
4. **About/Process** → Доверие
5. **Contact** → Действие (Telegram)

### Конверсионные точки:
- Header CTA (на всех страницах)
- Inline CTAs в контенте
- Footer CTA
- Dedicated Contact page
