Частые вопросы

Какая информация нужна для старта?

Опишите проект, задачу, желаемый стиль. Если есть референсы — отлично. Если нет — поможем определиться с направлением.

Сколько времени занимает работа?

Зависит от сложности. Аватар — 2-4 дня. Полный пакет — 5-7 дней. Обсудим сроки индивидуально.

Правки включены?

Да. Включены разумные правки по концепту и финальной версии.

Делаете анимированный дизайн?

Да. Анимированные аватары, баннеры, логотипы, эмодзи — все доступно.

Можно заказать полный пакет для проекта?

Конечно. Аватар + баннер + оформление соцсетей в единой визуальной системе — одна из наших сильных сторон.

Как происходит общение?

Через Telegram. Это быстро, удобно и позволяет оперативно обсуждать детали.

В каких форматах передаются файлы?

PNG, JPG для статики. GIF, MP4 для анимации. Исходники по запросу.
